Project Description
The purpose of this library is to enable easy use of the Audio/Video Shield.  This library will include functions which would allow the audio/video shield to work as a security camera. 

AV_Shield Library Overview
 - lib 
   - Library code where python code can be added on to be used within the main program. 
 - unit_test
   - test modules for the AV_shield library. 

Main Program Execution
 - main.py
   - stores main program to be executed with the library functions. 
 - run_test.py
   - testing code which initiates unit tests for code
